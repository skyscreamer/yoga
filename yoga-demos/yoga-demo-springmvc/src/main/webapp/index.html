Introduction

Enable "Relational REST" queries on your existing Java-based REST server.

  • Up to orders-of-magnitude performance improvements over multiple REST calls
  • Front-end and server programmers can develop independently
  • APIs that are browseable (HATEOAS)


The current version of Yoga is 0.9.0.

REST Selector Styles

Yoga supports two styles of REST field selector:

  • GData
  • LinkedIn

Supported Frameworks

Yoga can extend APIs built on:

  • Spring MVC
  • RESTEasy
