Introduction

Yoga provides GData- and LinkedIn-style data selectors for REST APIs.

  • Choose data fields at call-time
  • Much faster response times by reducing network calls
  • Allow clients to "build" their own API out of a simple entity structure
  • Simple plug-in to JAX-RS and SpringMVC RESTful implementations
  • Browsable APIs (HATEOAS)


The current version of Yoga is 0.9.4.

Supported Frameworks

Yoga can extend APIs built on:

  • Spring MVC
  • RESTEasy

Maven

        <dependency>
            <groupId>org.skyscreamer</groupId>
            <artifactId>yoga</artifactId>
            <version>0.9.4</version>
        </dependency>

And pick one of the integration packages:

        <dependency>
            <groupId>org.skyscreamer</groupId>
            <artifactId>yoga-springmvc</artifactId>
            <version>0.9.4</version>
        </dependency>

        <dependency>
            <groupId>org.skyscreamer</groupId>
            <artifactId>yoga-jaxrs</artifactId>
            <version>0.9.4</version>
        </dependency>
